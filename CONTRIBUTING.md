# Katkı Rehberi

Katkılar memnuniyetle karşılanır! :)

## 1. Issue açın
Değişiklik yapmadan önce bir issue açarak tartışma başlatın.

## 2. Fork → Branch → PR akışı
- Depoyu forkla
- Yeni bir branch oluştur (`feature/...`)
- Değişikliklerini yap
- PR gönder

## 3. Kod Kalitesi
- Temiz, anlaşılır ve belgeli kod yazın.
- Gereksiz karmaşıklıktan kaçının.
- Proje standartlarına uyun.

Şimdiden teşekkürler.
